# blync

[![Build Status](https://travis-ci.com/szechyjs/blync.svg?branch=master)](https://travis-ci.com/szechyjs/blync)

## macOS Requirements
```sh
brew install cmake
```

## Linux Requirements
```sh
apt-get install build-essential cmake libusb-1.0-0-dev
```

## Building/Install
```sh
mkdir build
cd build
cmake ..
make install
```

## Usage
```sh
blync [color]
```

### Colors
 - red
 - green
 - blue
 - magenta / purple
 - orange
 - cyan
 - yellow
 - white
 - off (any input other than above)
